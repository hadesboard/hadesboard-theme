== Description ==

HadesBoard is a minimal, fast, and SEO-friendly Persian portfolio theme with dark mode. Perfect for developers, freelancers, and creative professionals.

Features:
* Minimal dark design
* RTL support (Persian/Arabic)
* Custom Projects post type
* Full Customizer integration
* SEO-optimized with Schema.org markup
* Open Graph meta tags
* Performance optimized (no jQuery, minimal CSS)
* Gutenberg ready
* Translation ready

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate through Appearance > Themes
3. Go to Appearance > Customize to configure

== Customization ==

Navigate to Appearance > Customize > تنظیمات هادس‌بورد to:
* Add personal information
* Configure social links
* Customize section titles
* Show/hide sections
